# Component Docs

Use this directory for consumer-facing component usage notes.

Available docs:
- [Toast](./toast.md): programmatic toast service and viewport stack
- [SelectList](./select-list.md): single or multi selection with static or async data
- [DataTable](./data-table.md): sortable, selectable, paginated data tables
- [Paginator](./paginator.md): standalone pagination controls

Guidelines:
- Keep `README.md` at the package root short and focused on installation plus docs entrypoints.
- Put component examples and API notes in this folder instead of expanding the package root README.
- Document only consumer-facing inputs, outputs, behaviors, and migration notes.
